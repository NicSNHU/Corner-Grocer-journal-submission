# Corner-Grocer-journal-submission

This project uses industry best standards, declaring variables in one file and defining them in another. It practices Object Oriented Programming practices. This project titled Corner Grocer is a basic program that allows a user to visualize a display menu. The user can then select from 4 options from the main menu, in which after a decision is made, will return to the menu. Option 1 prompts the user to enter a grocery item and the program will return the item and the frequency of that item read from a text file of groceries, and their frequencies. Option 2 will will allow the user to see the entire list as well as a numeric amount for the frquency. Option 3 will allow the user to view all items read from the list and their frequencies however, it will visualize them as a histogram, or in other words, '*' symbol will appear for the amount of each item. Option 4 will allow the user to terminatet the program, or exit. This solves the problem of keeping track of inventory, through reading and writing to another file.
